This extension is directed for medium/small streamers
This chrome extension was intended for small/medium streamers that like to interact with the chat and they dont want to miss out on people that just arrived.

This works for any twitch channel but it has to be activated by checking the checkbox next to the chatbox.

This extesnion can be installed from: https://chrome.google.com/webstore/detail/riotgrrrl-twitch-chat-too/cjbnecgmlelpeajhmaafclchameiecma
